## 1. 产品概述

徽章设计器——一个让俱乐部会员亲手搭配专属身份徽章的在线工具。提供三种几何底板、图标叠加、代号文字、颜色/描边/纹理调节，以及四套主题包一键切换，设计完成后可导出图片带走。
- 目标用户：俱乐部会员，追求个性化表达与审美自主
- 核心价值：零门槛的创意设计体验，让每个人都能产出「我」的视觉符号

## 2. 核心功能

### 2.1 用户角色
| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 会员 | 无需注册 | 使用全部设计功能并导出 |

### 2.2 功能模块
1. **设计器页面**：徽章画布预览区 + 控制面板（底板/图标/代号/颜色/描边/纹理/主题/撤销重做/导出）

### 2.3 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 设计器页面 | 徽章画布 | 实时渲染圆形/六边形/不规则碎片底板，叠加图标与代号文字，应用颜色/描边/纹理 |
| 设计器页面 | 底板选择 | 三种几何形状切换，选中后即时刷新画布 |
| 设计器页面 | 图标叠加 | 从内置图标库选择图标，拖拽定位（不可超出底板范围） |
| 设计器页面 | 代号文字 | 输入2-8个字符的代号，实时渲染到徽章上 |
| 设计器页面 | 颜色调节 | 底板填充色、描边色、文字色独立选取 |
| 设计器页面 | 描边调节 | 描边宽度与样式（实线/虚线） |
| 设计器页面 | 纹理叠加 | 噪点/网格/渐变三种纹理，可开关和调参 |
| 设计器页面 | 主题包 | 赛博/手工/暗房/磁带四套主题一键切换配色与纹理 |
| 设计器页面 | 撤销/重做 | 至少支持20步撤销与重做 |
| 设计器页面 | 导出 | 校验代号与必填内容后导出PNG图片 |

## 3. 核心流程

用户进入设计器 → 选择底板形状 → 调整颜色/描边/纹理 → 叠加图标 → 输入代号 → （可选）切换主题包 → 撤销/重做调整 → 校验 → 导出图片

## 4. 用户界面设计

### 4.1 设计风格
- **主色**：深灰/炭黑为底色（#1a1a2e），搭配铜金/琥珀色点缀（#d4a574）
- **辅助色**：暖白（#f0e6d3）、墨绿（#2d4a3e）
- **按钮风格**：微圆角矩形，hover时发光边框
- **字体**：显示字体使用 Playfair Display（徽章代号），UI字体使用 DM Sans（面板文字）
- **布局风格**：左侧画布预览区 + 右侧控制面板，桌面优先
- **图标风格**：Lucide 线性图标

### 4.2 页面设计概览
| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 设计器页面 | 画布区域 | 居中大尺寸SVG画布，深色背景，徽章居中渲染，周围留白呼吸感 |
| 设计器页面 | 底板选择 | 三个缩略图按钮横向排列，选中态铜金描边 |
| 设计器页面 | 图标选择 | 网格弹窗，hover放大效果 |
| 设计器页面 | 代号输入 | 输入框 + 字数计数器（x/8） |
| 设计器页面 | 颜色选择 | 原生颜色拾取器 + 预设色块 |
| 设计器页面 | 描边控制 | 滑块（宽度）+ 切换按钮（实线/虚线） |
| 设计器页面 | 纹理控制 | 三种纹理标签页切换 + 开关 + 参数滑块 |
| 设计器页面 | 主题包 | 四个主题卡片横向排列，hover展示预览 |
| 设计器页面 | 撤销/重做 | 工具栏图标按钮，禁用态灰色 |
| 设计器页面 | 导出按钮 | 底部固定，铜金色主按钮，点击触发校验 |

### 4.3 响应式
- 桌面优先（1440px+），画布与面板左右布局
- 平板（768-1439px），画布缩小，面板折叠为底部抽屉
- 移动端（<768px），画布置顶，面板全屏抽屉

### 4.4 动效
- 画布切换底板时平滑过渡（scale + opacity）
- 主题切换时全局颜色渐变过渡
- 控制面板展开/折叠动画
- 导出成功时徽章弹跳动画
