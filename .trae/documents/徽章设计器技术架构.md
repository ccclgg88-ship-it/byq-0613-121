## 2. 技术说明
- 前端：React@18 + TypeScript + Tailwind CSS + Vite
- 初始化工具：vite-init
- 状态管理：Zustand（徽章状态 + 撤销/重做历史）
- 画布渲染：SVG（实时预览）+ Canvas API（导出）
- 后端：无
- 数据库：无

## 5. 关键实现策略

### 5.1 SVG 画布渲染
- 底板形状使用 SVG `<path>` / `<circle>` / `<polygon>` 绘制
- 纹理通过 SVG `<filter>` 实现（feTurbulence 生成噪点、feImage 叠加网格、linearGradient 渐变）
- 图标使用内联 SVG symbol，通过 `<use>` 引用
- 代号使用 SVG `<text>` 元素，居中定位

### 5.2 撤销/重做
- Zustand 中维护 past/future 栈，每次状态变更时将当前状态推入 past
- 限制 past 最大长度为 30（超出 20 步余量）
- 撤销：present → future，past 末尾 → present
- 重做：present → past，future 末尾 → present

### 5.3 导出
- 将 SVG 转为 Canvas：使用 Image 加载 SVG data URL，绘制到 Canvas 后导出 PNG
- 导出前校验：codename 长度 2-8 字符、图标已选择
- 校验失败时在 UI 上显示红色提示

### 5.4 图标范围约束
- 图标中心坐标默认为底板中心，允许拖拽偏移
- 拖拽时实时计算图标边界是否超出底板 clip-path 区域
- 超出时回弹到最近合法位置
